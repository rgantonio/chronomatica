# Brain-Computer Interface (BCI)
- In this directory, we do a study of a simple BCI case of seizure detection.
- This meant to understand and profile kernels that run in a BCI pipelines. 

## File List
- `seize_detect_details.ipynb`: Details of the seizure detection pipeline.
- `seize_detect.py`: Original seizure detection code without timing profiling.
- `seize_detect_profile.py`: Upgraded seizure detection code with timing profiles.
